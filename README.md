2D指向性エネルギー兵器（DEW）シミュレーター

本プロジェクトは、指向性エネルギー兵器（Directed Energy Weapon: DEW）におけるレーザー光の伝搬と、ターゲット装甲板への熱伝達・破壊（アブレーション）プロセスを物理学的に計算し、リアルタイムに可視化するインタラクティブなブラウザアプリです。

HTML5 Canvas と JavaScript (ES6) を用いて、ビジュアルと学術的数値計算を1ファイルで融合させています。

🚀 主な機能

リアルタイム熱収支物理シミュレーション

レーザー吸収による温度上昇だけでなく、対流冷却や熱伝導、ステファン・ボルツマン則に基づく熱放射損失（$dQ$）をフレームごとに計算。

ガウシアンビーム回折限界（Gaussian Beam Diffraction）の再現

ターゲット位置（500m固定）に対し、システムの「ビーム焦点距離」を変化させることで、ビームスポット径が変化。回折限界によるビーム拡散と、それに伴うエネルギー密度（$\text{W/cm}^2$）の変動を再現。

複数素材データベースの搭載

高炭素鋼、アルミニウム（Al 6061）、チタン合金（Ti-6Al-4V）、石英ガラス（Quartz）の4素材を搭載。それぞれの比熱、熱伝導率、密度、融点、レーザー吸収率の違いがシミュレーション結果に直結。

ダイナミック・ビジュアルエフェクト

焦点距離に同期したビーム形状の変形、ターゲット部の温度カラーマップ、温度上昇に伴うオレンジ～白熱の熱輝光（発光）、スパークパーティクル、融解貫通（穴あき）プロセスを Canvas 2D で美しく描写。

リアルタイムトレンドグラフ

経過時間とターゲット温度の推移を時間グラフに描画。融点に達した瞬間に融解フェーズへと移行する様子が視覚的に理解可能。

📐 シミュレーターのコア物理モデル

1. ガウシアンビームのスポット径計算

レーザーは平行光線であっても波の性質（回折）により必ず拡散します。ターゲット距離 $z$、焦点位置 $z_f$、初期ビーム半径 $w_0$、レーザー波長 $\lambda$ とすると、ターゲット上でのビーム半径 $w(z)$ は以下のように求められます。

$$w(z) = w_0 \sqrt{1 + \left(\frac{\lambda (z - z_f)}{\pi w_0^2}\right)^2}$$

これに基づき、焦点（$z_f = 500\text{m}$）がターゲットと完全に一致したときスポット径が最も細くなり、エネルギー密度（破壊力）が最大化します。

2. 熱平衡方程式

ターゲットに流入する熱量 $Q_{\text{in}}$ と、逃げていく熱量 $Q_{\text{out}}$ （対流・放射）の差分から、温度変化 $dT$ を計算します。

$$dQ = Q_{\text{in}} - Q_{\text{out}}$$

$$Q_{\text{in}} = P \cdot \eta \cdot dt$$

$$Q_{\text{out}} = \left[ h \cdot A \cdot (T - T_{\infty}) + \epsilon \cdot \sigma \cdot A \cdot (T^4 - T_{\infty}^4) \right] \cdot dt$$

$$dT = \frac{dQ}{m \cdot C_p}$$

$P$: レーザー出力 $(\text{W})$

$\eta$: 素材のレーザー吸収率 $(0 \sim 1)$

$h$: 対流熱伝達率

$A$: スポット面積

$T$: 現在の温度、 $T_{\infty}$: 周囲温度

$\epsilon$: 放射率、 $\sigma$: ステファン・ボルツマン定数

$m$: 相互作用部分の推定質量

$C_p$: 素材の比熱容量 $(\text{J/(kg}\cdot\text{K))}$

🛠 動作環境・セットアップ

このシミュレーターは、外部アセット（CSS/アイコンなど）をCDN経由で読み込んでいるため、インターネットに接続された環境であれば環境構築をすることなく動作します。

簡単起動（シングルファイル）

laser_weapon_simulator.html を任意の場所に保存します。

ファイルをダブルクリック（またはブラウザにドラッグ＆ドロップ）して開きます。

開発環境での展開（Vite / Vanilla TS等）

TypeScript や Bundler（Viteなど）環境で動作させる場合：

src/main.ts にスクリプト部分（script タグ内のコード）を移植。

index.html に UIレイアウト（body 内の HTML 構造）および Tailwind CSS / Font Awesome のリンクを設定。

npm run dev で開発サーバーを起動。

🎮 シミュレーションの楽しみ方（検証例）

焦点とエネルギー密度の関係:
焦点をターゲット位置である 500m にピッタリ合わせると、スポット径が最小（約 0.20mm 以下）になり、超高エネルギー密度によって一瞬で加熱されます。逆に焦点を 100m や 2000m にずらすと、ビームが拡散し、高出力でも温度がほとんど上がらなくなります。

素材による耐熱性の違い:

チタン合金: 熱伝導率が非常に低いため、レーザーが当たった部分の熱が逃げにくく、極めて急速に局部加熱が起こり溶融します。

アルミニウム: 熱伝導率が高いため、周囲に熱が逃げてしまい、チタンよりも融点が低いにもかかわらず、貫通までに時間がかかる場合があります。
